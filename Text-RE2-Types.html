Compile options

data CompileOption Source

Options available when compiling a regex.

Defaults (in parentheses) are according to http://code.google.com/p/re2/source/browse/re2/re2.h. The Haskell binding does not enforce these.

If multiple options with the same constructor are given in a list, the last one has priority.

Constructors

PosixSyntax Bool

(False) Restrict regexps to POSIX egrep syntax.

LongestMatch Bool

(False) Search for longest match, not first match.

LogErrors Bool

(True) Print errors to stderr.

Literal Bool

(False) Interpret string as literal, not regexp.

NeverNewline Bool

(False) Never match \n, even if it is in regexp.

CaseSensitive Bool

(True) Match is case-sensitive.

PerlClasses Bool

Allow Perl's \d \s \w \D \S \W even in POSIX mode.

WordBoundary Bool

Allow Perl's \b \B even in POSIX mode.

OneLine Bool

Match ^ and $ only at beginning and end of text even in POSIX mode.

MaxMemory Int

Memory limit, in bytes.

data MatchOptions Source

Options available when matching with a regex.

For default options use defMatchOptions, or import Data.Default and use def.

anchor :: Anchor

captureLimit :: Maybe Int

If Just n, only store the first n capturing groups. This includes the whole-match group, so Just 1 returns only that group, and Just 0 is a Boolean match only.

errFragment :: ByteString

A fragment of the regex relating to the error. When the regex was given as String or Text, this fragment will be UTF-8.
